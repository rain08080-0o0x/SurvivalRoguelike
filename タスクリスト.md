# 奈落塔 プロトタイプ タスクリスト

## このファイルの扱い

このファイルは、奈落塔プロトタイプの作業タスクを管理するための一覧です。
作業が増えた、完了した、優先度が変わった場合は、このファイルへ追記または更新します。

更新時の詳細ルールは [ドキュメント更新ルール.md](ドキュメント更新ルール.md) を参照します。

## 優先度 高

- [ ] 実機操作確認
  - [ ] WASDがカメラ基準で自然に動くか確認する。
  - [ ] Shift長押しで走れるか確認する。
  - [ ] Shift短押しでステップできるか確認する。
  - [ ] Spaceジャンプが違和感ないか確認する。
  - [ ] ロープ中A/Dで離脱できるか確認する。

- [ ] 3D表示の確認

  - [ ] カメラ距離、FOV、角度が探索ゲームとして見やすいか確認する。
  - [ ] 深度が下方向に見えるか確認する。

- [ ] 操作と表示のズレ修正
  - [ ] 移動方向と画面上の移動方向を確認する。
  - [ ] 攻撃方向と表示ラインを確認する。
  - [ ] ステップ方向とカメラ基準移動を確認する。
