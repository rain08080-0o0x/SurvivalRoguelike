# 奈落塔 開発記録帳

## このファイルの扱い

このファイルは、奈落塔プロトタイプで実施した作業を日付別に記録するためのログです。
今後、仕様変更、実装、修正、調査、ビルド確認を行った場合は、このファイルへ追記します。

更新時の詳細ルールは [ドキュメント更新ルール.md](ドキュメント更新ルール.md) を参照します。

## 2026-05-12

### プロジェクト作成

- `C:\HAL\個人制作\NarakuProto` にプロトタイプ用プロジェクトを作成。
- 既存プロジェクト `C:\HAL\4-2after\DX22\Program\DX22_Project` から、DirectX、ImGui、Input、Model、Assimp、DirectXTex周辺を含む構成を拝借。
- `build_debug_x64.ps1` による Debug x64 ビルドを確認。

### プロトタイプシーン追加

- `SceneNarakuProto.h` を追加。
- `SceneNarakuProto.cpp` を追加。
- `SceneManager` に `SCENE_NARAKU_PROTO` を追加。
- 起動時に奈落塔プロトタイプシーンへ入るように調整。

### 基本ループ実装

- プレイヤー移動、走り、ステップ、ジャンプを実装。
- スタミナを実装。
- 重量システムを実装。
- 上昇負荷のプロトタイプ用ルールを実装。
- 採掘ポイントを10か所配置。
- 4級旧器の取得、所持、破棄、売却を実装。
- 帰還、売却、再潜行を実装。
- 死亡リザルトを実装。
- 地図ピンの設置/削除を実装。

### UI

- ImGuiの文面を日本語化。
- HUD、所持品、旧器発見確認、帰還/死亡リザルト、地図ピン用UIを実装。

### 入力調整

- Shift短押しでステップ、Shift長押しで走りに分岐する処理を調整。
- Shiftの判定を左右Shiftの物理キーで見るように調整。
- 走りが反応しない問題を修正。
- ステップが反応しない問題を修正。
- 単押し判定のしきい値を確認し、0.18秒基準で運用。

### 3Dデバッグ描画

- 既存の `Geometory` を使って、プレイヤー、敵、採掘ポイント、ロープ、旧器、帰還地点、ピンを3D表示。
- 斜め見下ろしカメラを追加。
- 3D表示時に箱/球のワールド行列が崩れる問題を修正。
- `Main.cpp` の共通デバッググリッドが奈落プロトと重なる問題を修正。
- カメラ基準移動を導入。
- 左右移動が反転していた問題を修正。

## 2026-05-13

### ロープ操作修正

- ロープ使用中にスタミナが0になると、昇降できず操作不能に近い状態になる問題を確認。
- ロープ使用中に `A` / `D` を押すと、スタミナがなくてもロープから離脱できるように修正。
- ロープから離脱した時、少し横へずらして再度ロープ判定に吸われにくくした。
- ロープに掴まって停止しているだけなら、スタミナが回復するように修正。
- HUDの操作説明に `ロープ中A/D 離脱` を追加。
- Debug x64 ビルド成功を確認。

### ドキュメント整備

- `.sln` と同じディレクトリに企画草案、タスクリスト、開発記録帳、ドキュメント更新ルールを作成。
- 今後の変更時に、関連ドキュメントへ追記する方針を明文化。

### 半透明床・仮地形

- 既存の `Sprite` を使い、1x1白テクスチャにRGBA色を掛けて半透明床を描画する方式を追加。
- 地表全体を薄い半透明床として表示。
- ロープ周辺に下層床を仮配置。
- 崖穴の目印として暗い半透明面を追加。
- 画像素材なしで動く構成にしたため、必要になったら画像テクスチャへ差し替え可能。
- Debug x64 ビルド成功を確認。

### 地形判定・一段下の床

- `FloorRegion` を追加し、地上床と一段下の床を移動判定用データとして持つように変更。
- 通常移動、ステップ、ノックバックが現在深度の床外へ出ないように制限。
- `RopePoint` を追加し、ロープに上端深度と下端深度を持たせた。
- ロープ昇降中の深度を上端から下端の範囲に制限。
- ロープから横へ離れる時は、現在深度に床がある場合だけ離脱できるように変更。
- 半透明床の描画を、実際の床判定データから生成する方式に変更。
- Debug x64 ビルド成功を確認。
